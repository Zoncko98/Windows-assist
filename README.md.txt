# AI Assistant

This is an AI assistant built with Python that utilizes machine learning models to generate responses, process speech input, and interact through a graphical user interface (GUI). It uses popular libraries like Hugging Face Transformers, PyTorch, and others for natural language processing and voice recognition.

## Features
- Speech input and output
- Text-based interaction via GUI
- Fine-tuning models for specific tasks
- Web scraping (optional)
- System resource monitoring
